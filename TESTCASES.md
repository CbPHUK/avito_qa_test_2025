# TESTCASES.md — Тест-кейсы для API микросервиса объявлений

## Позитивные сценарии

1. Создание объявления с валидными данными
   - Ожидаемо: 200 OK, возвращается ID объявления.

2. Получение объявления по валидному ID
   - Ожидаемо: 200 OK, возвращаются все поля объявления.

3. Получение всех объявлений по sellerId
   - Ожидаемо: 200 OK, список объявлений продавца.

4. Получение статистики по объявлению
   - Ожидаемо: 200 OK, объект со статистикой (likes, viewCount, contacts).

## Негативные сценарии

1. Создание объявления без обязательных полей
   - Ожидаемо: 400 Bad Request.

2. Получение объявления по несуществующему ID
   - Ожидаемо: 404 Not Found.

3. Получение статистики по несуществующему ID
   - Ожидаемо: 404 Not Found.

4. Удаление по несуществующему ID
   - Ожидаемо: 404 Not Found.

## Граничные сценарии

1. Цена = 0
   - Ожидаемо: проходит.

2. Максимальные значения полей
   - Ожидаемо: проходит.

3. Минимальные значения (пустые строки, где разрешено)
   - Ожидаемо: проходит или 400.
